# Muhtar_bot
A subreddit monitoring and metric collecting bot that can also moderate.

```
python run_muhtar_bot.py
```

## Installation

```
pip install -r requirements.txt
```

# DB UI

See, https://github.com/huggingface/Mongoku

```
    npm install -g mongoku
    mongoku start
```
